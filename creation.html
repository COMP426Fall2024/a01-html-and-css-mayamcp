Mayan Mythology

Creation

The Mayan Creation story, told in

Phase Description
1st Attempt The gods create the world from the void but fail to create humans.
2nd Attempt The gods create animals, but they cannot worship the gods.
3rd Attempt The gods create humans from mud, but they dissolve.
4th Attempt Humans are created from maize, becoming the successful creation.
